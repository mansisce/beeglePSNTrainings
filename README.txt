Utensil Cleaning — GitHub Pages Package

Contents:
- index.html
- images/ (illustrative PNGs)
- audio/ (placeholder WAV files for each language & section)

Deploy to GitHub Pages:
1. Create a GitHub repo.
2. Copy all files into the repo root.
3. Commit & push to main branch.
4. Settings → Pages → Deploy from main branch.

Replace placeholder audio with TTS:
- Generate WAV/MP3 using TTS (Google/AWS/Azure or free tools).
- Use filenames like: en_prep.wav, hi_prep.wav, bn_prep.wav, ta_prep.wav, etc.
- Place in audio/ directory and re-deploy.
